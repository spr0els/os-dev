# Preparation
The build script expects the directories of `binutils`, `gdb` and `gcc` containing the respective sources in this directory.

Set the variables pointing to these directories in the build script accordingly.

# Building
The build script should take care of building and installing these tools automatically.
You may want to customize the exported environment variables.

For more information regarding building the tools refer to OS Dev's GCC Cross-Compiler Page (https://wiki.osdev.org/GCC_Cross-Compiler).
